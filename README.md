## 数据集构建
对公开数据集loghub根据RFC 5424 (Syslog Protocol) 定义的 Facility 标准以及 NIST SP 800-92 (计算机安全日志管理指南)，将提供的 118 个日志模板归纳为以下 7 个核心语义类别和3个异常类别。 
## 模型
